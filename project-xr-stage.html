Project

실시간 XR 스테이지 구현: Unreal Engine & nDisplay 파이프라인 구축

이질감 없는 가상 환경 구현을 위한 실시간 렌더링 최적화 및 디바이스 동기화 프로젝트

Summary

nDisplay 및 Disguise 솔루션을 활용하여 오프라인 무대와 가상 환경이 실시간으로 상호작용하는 XR 콘텐츠 제작.

Tech Stack

  • Unreal Engine 5.2
  • Disguise (d3)
  • nDisplay
  • Maya

Key Achievement

  • nDisplay 설정을 통한 멀티 디스플레이 렌더링 최적화: 엔진 내부의 비주얼이 실제 무대 LED에 왜곡 없이 송출되도록 파이프라인 설계.
  • UE 5.2 기반 고퀄리티 배경 레벨 디자인: 실시간 렌더링 환경에서도 프레임 드랍을 방지하면서 실사 지향적 비주얼 구현.
  • 디스가이즈 미디어 서버 워크플로우 마스터: 미디어 서버와 엔진 간의 데이터 송수신 및 캘리브레이션 프로세스 수행.
